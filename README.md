# Ceng112 OPP in Python Practice Tasks

Aim of these tasks is learning OOP step by step and getting high score in CENG112 quiz.

## Tasks Overview

- **task1.py**: Create a Student class with name and age attributes and print student info.
- **task2.py**: Create a Rectangle class to calculate area and perimeter.
- **task3.py**: Create a BankAccount class with deposit, withdraw, and balance check methods.
- **task4.py**: Create a TV class to control channel and volume settings.
- **task5.py**: Demonstrate inheritance and polymorphism with Animal, Dog, and Cat classes.
- **task6.py**: Create Person hierarchy with Student and Teacher classes, including method overriding and equality checks.
- **task7.py**: Create Student class with course management functionality.
- **task8.py**: Implement a TextEditor with undo and redo functionality using a Stack.
- **task9.py**: Implement a basic Queue data structure.
